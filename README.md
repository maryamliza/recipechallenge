
# RecipeApp

Es una aplicación en la cual podrás encontrar recetas y el detalle de ellas para prepararlas.



## Tecnologías usadas:
- Kotlin
- MVVM
- ViewModel
- Coroutines
- Navigation Component
- ViewBinding
- Retrofitz
- Koin
- Glide
- Maps SDK
- Mockk
